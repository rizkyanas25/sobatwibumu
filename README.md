```

## 🎨 Features

- **The Ultimate Playlist Hub** — Quick access to curated Anisong playlists
- **Racun Section (Referrals)** — Showcase of personal setup essentials and figures
- **Dynamic Glassmorphism** — Sleek, modern blurred backdrop aesthetic
- **Fully Responsive** — Optimized for all devices, tailored for mobile
- **Performance Optimized** — Built with Next.js 15

## 📁 Project Structure

```
sobatwibumu/
├── app/                 # Next.js App Router
│   ├── globals.css      # Design tokens & global styles

│   └── page.tsx         # Main page
├── components/          # Reusable UI components
│   ├── site-header.tsx
│   ├── referrals-section.tsx
│   └── ...
├── data/                # Content data (JSON/TS)

## 🎯 Design System

### Colors

- **Background**: `#050505` (Deep Dark)
- **Foreground**: `#f6f1e8` (Soft White)
- **Gradients**: Subtle Sky Blue & Rose highlights

### Typography

- **Heading**: Avenir Next Condensed, Trebuchet MS
- **Body**: Avenir Next, Segoe UI

### Principles

- High-end glassmorphism panels
- Extremely smooth scroll navigation
- Clean and uncluttered presentation
